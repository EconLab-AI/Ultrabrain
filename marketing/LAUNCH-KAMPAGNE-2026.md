### MONTAG - Feature: Lifecycle Hooks
```
Claude Code has 5 lifecycle hooks.

Most plugins use 1 or 2.

UltraBrain uses ALL 5:

SessionStart → inject relevant memories
UserPromptSubmit → enrich with context
PostToolUse → capture what happened
Stop → generate session summary
SessionEnd → compress & embed

This is how you build AI that learns.
```

```bash

  -vf "crop=800:800:140:400,scale=480:480" \
```
